1

•

Create a new file in the _datapacks folder in the Data Pack Repository - here


2

•

Name the file pack-id.md - pack-id is what will show as the URL of your pack on this site ({{ site.url}}/data-packs/pack-id)


3

•

Fill in the file contents with this template... 

---
layout: data_pack

title: "PACK_TITLE"
supported_versions: [ 1.13, 1.14, 1.15, 1.16 ]
authors: "A, LIST, OF, AUTHORS"
---

Write a full, html/markdown-supported description here!

4

•

Scroll down and click Propose new file


5

•

Confirm by clicking Create pull request, name the pull request [Data Pack] Your Data Pack Name, and click Create pull request again


6

•

You're done! Just wait for your pack to be approved
